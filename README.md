# Demo Intermedio

Este es el proyecto "demo-intermedio", un menú digital para restaurantes con diseño premium y optimización para escritorio y móvil.

## Características
- Diseño responsivo y moderno.
- Menú interactivo.
- Sistema de pedidos vía WhatsApp.
- Optimización de layout para pantallas grandes.

## Instalación
1. Clona el repositorio.
2. Abre `index.html` en tu navegador o sírvelo mediante un servidor local como XAMPP.
